STOCKINFO
Stock-Codes API About Donate
GET /download_csv
Download stock price data as CSV.
Example: /download_csv?ticker=AAPL
GET /download_pdf
Download a PDF report summarizing stock info.
Example: /download_pdf?ticker=TSLA
GET /report
View detailed HTML analysis of stock data.
Example: /report?ticker=GOOGL
GET /raw
Access raw stock data in JSON format.
Example: /raw?ticker=MSFT
